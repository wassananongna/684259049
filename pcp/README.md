# Electricity Calculator

A Java program to calculate electricity bills based on tiered consumption rates.

## Pseudocode (English)
1. **START**
2. **INPUT** `units` (kWh)
3. **SET** `SERVICE_CHARGE` = 50.0
4. **IF** `units` <= 150:
   - `bill` = `units` * 3.50
5. **ELSE IF** `units` <= 400:
   - `bill` = (150 * 3.50) + ((`units` - 150) * 4.20)
6. **ELSE**:
   - `bill` = (150 * 3.50) + (250 * 4.20) + ((`units` - 400) * 5.00)
7. **SET** `total` = `bill` + `SERVICE_CHARGE`
8. **DISPLAY** `units`, `bill`, `SERVICE_CHARGE`, and `total`
9. **END**

## รหัสเทียม (ภาษาไทย)
1. **เริ่มต้น**
2. **รับค่า** `หน่วยการใช้งาน` (kWh)
3. **กำหนด** `ค่าบริการ` = 50.0 บาท
4. **ถ้า** `หน่วยการใช้งาน` <= 150:
   - `ค่าไฟฟ้า` = `หน่วยการใช้งาน` * 3.50
5. **ถ้า** `หน่วยการใช้งาน` <= 400:
   - `ค่าไฟฟ้า` = (150 * 3.50) + ((`หน่วยการใช้งาน` - 150) * 4.20)
6. **มิฉะนั้น**:
   - `ค่าไฟฟ้า` = (150 * 3.50) + (250 * 4.20) + ((`หน่วยการใช้งาน` - 400) * 5.00)
7. **กำหนด** `ยอดรวม` = `ค่าไฟฟ้า` + `ค่าบริการ`
8. **แสดงผล** `หน่วยการใช้งาน`, `ค่าไฟฟ้า`, `ค่าบริการ` และ `ยอดรวม`
9. **จบการทำงาน**
