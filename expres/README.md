# ระบบคำนวณค่าจัดส่งสินค้า (Shipping Cost Calculator)

โปรแกรมนี้เป็นระบบที่ช่วยคำนวณค่าจัดส่งสินค้าตามน้ำหนักและประเภทการจัดส่ง โดยมีระบบคิดค่าบริการเพิ่มเติมสำหรับบริการด่วน และส่วนลดสำหรับสมาชิก VIP

## คุณสมบัติเด่น

- คำนวณค่าจัดส่งตามน้ำหนักสินค้า
- รองรับบริการจัดส่งแบบมาตรฐานและแบบด่วน
- มีระบบส่วนลดสำหรับสมาชิก VIP
- อินเตอร์เฟซที่ใช้งานง่ายพร้อมเมนูภาษาไทย
- การตรวจสอบข้อมูลนำเข้าเพื่อป้องกันข้อผิดพลาด

## อัตราค่าจัดส่ง

### บริการมาตรฐาน (Standard):
- น้ำหนัก 0–1 กิโลกรัม = 40 บาท
- น้ำหนักมากกว่า 1–3 กิโลกรัม = 60 บาท
- น้ำหนักมากกว่า 3–5 กิโลกรัม = 80 บาท
- น้ำหนักมากกว่า 5 กิโลกรัม = 100 บาท

### บริการด่วน (Express):
- น้ำหนัก 0–1 กิโลกรัม = 60 บาท
- น้ำหนักมากกว่า 1–3 กิโลกรัม = 90 บาท
- น้ำหนักมากกว่า 3–5 กิโลกรัม = 120 บาท
- น้ำหนักมากกว่า 5 กิโลกรัม = 150 บาท

### บริการเพิ่มเติม:
- บริการด่วน (Express) เพิ่มค่าบริการ 30 บาท
- สมาชิก VIP ลดค่าจัดส่ง 20%

## การใช้งาน

1. คอมไพล์โปรแกรมโดยใช้คำสั่ง:
   ```
   javac -encoding UTF-8 ShippingCalculator.java
   ```

2. รันโปรแกรมโดยใช้คำสั่ง:
   ```
   java ShippingCalculator
   ```

3. ป้อนน้ำหนักของสินค้า
4. เลือกประเภทบริการ (มาตรฐานหรือด่วน)
5. เลือกสถานะสมาชิก (VIP หรือไม่)
6. โปรแกรมจะแสดงรายละเอียดค่าจัดส่งพร้อมราคาสุดท้าย

## โครงสร้างไฟล์

- `ShippingCalculator.java` - ไฟล์หลักที่ประกอบด้วยคลาสสำหรับคำนวณค่าจัดส่งและเมทอดหลัก
- `algor.md` - เอกสารอัลกอริทึมพร้อมแผนภาพลำดับงานและรหัสเทียมภาษาอังกฤษ
- `algor-th.md` - เอกสารอัลกอริทึมพร้อมแผนภาพลำดับงานและรหัสเทียมภาษาไทย

## เงื่อนไขของระบบ

- น้ำหนักสินค้าต้องมากกว่า 0 กิโลกรัม
- ระบบรองรับการคำนวณค่าจัดส่งหลายรายการในหนึ่งรอบการทำงาน
- ระบบตรวจสอบข้อมูลนำเข้าเพื่อป้องกันค่าที่ไม่ถูกต้อง
